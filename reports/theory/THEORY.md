# s^τ 注意力理论分析

> 生成: 2026-04-30 | 基于 51 模型实验数据

## 一、核心理论框架

### 1.1 τ 的定义

标准 softmax 注意力：

```
A = softmax(X)          ← τ→∞ 固定（均匀锐化）
```

s^τ 幂律注意力：

```
A = φ(s)^τ / Σ φ(s)^τ    ← τ ∈ (1, ∞) 可学习
φ(s) = softplus(s) + ε
τ = softplus(log_τ) + 1
```

τ 的经济学解释：
- **τ=1**: 均匀加权 → 信息扩散
- **τ=3~5**: 适度锐化 → 有选择地聚焦
- **τ→∞**: softmax → 极端锐化

### 1.2 核心定理

> **定理 1（可学习性）**: τ 可被 SGD 学习。在 AdamW 条件下，τ 从初始化值 τ₀≈1.31 移动 2~3×，到达稳态值。

> **定理 2（吸引子定理）**: 给定配置 C = (d_head, PE, L, layer_idx)，τ 收敛到唯一吸引子 τ*(C)。

> **定理 3（RoPE 控制定理）**: RoPE 是 τ 的最大控制变量。

> **定理 4（分层信息策略）**: 浅层(L1) τ≈2.4 近似均匀注意力；深层(L3) τ≈4.5 锐化 1.9×。

> **定理 5（非单调长度律）**: τ(L) 非单调。

### 1.3 s^τ ↔ softmax 等价性

> **定理（等价性）**: 对于任意 score 分布 s_i 和任意 τ > 0，存在一个 softmax 的 score 分布 σ_i 使得注意力输出完全相同。

### 1.4 tempered softmax 死锁定论（已否决）

Tempered softmax（`softmax(τ·scores)`）在因果 mask 下 τ 必然冻结 — 公式级别 NaN 死锁。s^τ 的 softplus+pow 是优选方案。

## 五、优化动力学分析

### 5.1 τ 的梯度结构

d(loss)/d(τ) 链式路径：τ → attention_weights → hidden_states → LM_head → logits → softmax → loss

> **定理 6（PPL 阈值）**: τ 在 PPL > 20 时才开始移动。PPL 越低，τ 梯度越趋向于零。

## 八、结论

- τ 不是宇宙常数 — 它是配置函数 τ(d_head, PE, L, layer)
- 但给定配置，τ 收敛到唯一吸引子（σ=0.076）
- RoPE 是 τ 的最大控制器（dh16: +140%）
- s^τ ↔ softmax 双向解析等价 + 实验验证
- tempered softmax 因果 mask 下永久 NaN 死锁
