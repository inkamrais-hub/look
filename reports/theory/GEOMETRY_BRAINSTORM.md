# ULNP 几何头脑风暴 — 能玩的结构远不止李群

> 2026-05-30 · 整活专用 · 不保证有用, 保证有趣

## 1. 分形残差 — 幂律天然对应

O(2) 旋转的 cosθ/sinθ 是无特征尺度的平滑变换。但自然语言是分形的——Zipf 定律、长尾分布。

**想法**: 残差用分形插值替代线性旋转

```
h' = h · D_α(t) + update · (1 - D_α(t))
```

其中 D_α(t) 是分形维数 α 的魔鬼阶梯函数 (Cantor function)。不同 α 对应不同的记忆衰减模式:
- α=1: 欧氏空间，标准残差
- α→0: 近 Cantor 集，极端稀疏记忆 (只保留几个"骨架"token)
- α≈log2/log3: 经典 Cantor 三分集

每个指令可以有自己的 α_k，Router 学会选。

## 2. 热带几何残差 — Max-Plus 代数

乘法 → 加法，加法 → max。Transformer 的 softmax 本质是 log-sum-exp，而 max 是它的热带极限。

```
h'[i] = max( h[i] + log_cos, update[i] + log_sin )
       - logsumexp( log_cos, log_sin )  ← 归一化回正常空间
```

热带版本的注意力: `max_j (Q·K_j + V_j)` — 整片论文等着写。

ULNP 当前只用 Linear Program (标准代数)，热带 Program 是天然对偶。

## 3. 八元数残差 — 非结合代数

U(1) 是复数 (2D)，SU(2) 是四元数 (4D)。再往下是八元数 (8D) — **非结合的**。

这意味着 `(h · instr_a) · instr_b ≠ h · (instr_a · instr_b)`。指令的**执行顺序**开始敏感！

在 ULNP 里 Router 输出指令权重，但指令之间是无序积累的 (sum over k)。八元数天然要求我们说清"先做哪条、后做哪条"。
这个非结合性可能正好对应**语法树的非线性组合**。

```
d_model 必须是 8 的倍数
h = [a,b,c,d,e,f,g,h] → 八元数
指令 = 八元数旋转 (8×8 实矩阵保范数, 但乘法结构非结合)
```

## 4. p-adic 残差 — 超距几何

p-adic 数的几何不是"近大远小"而是**树状分层**：两个数之间的距离由它们的 p-进制展开决定。越深的位相同 → 越"近"。

在序列建模中: "相同词根"的 token 在 p-adic 度量下天然聚成子树，而"不同词根但同为名词"的在浅层相聚。

```
替代 LayerNorm: 用 p-adic 范数 ||h||_p 正则化
替代欧氏距离: 用 p-adic 超距决定注意力范围
```

论文标题: *p-adic ULNP: Ultrametric Token Mixing*

## 5. 时晶残差 — 周期性结构涌现

时间晶体在基态有自发周期性——没有外部输入也会振荡。对应到序列处理: 某些残差流天然有**固定节律**。

```
h' = h · cos(θ + ω·t) + update · sin(θ + ω·t)
```

ω 是可学习的振荡频率。不同层、不同指令可以锁定到不同频率，形成**频率复用**——有点像神经科学里的 theta-gamma 跨频耦合。

如果 ω_k 是分数倍的基频，就自动有了**节律编码**：音节级(r=1)、词级(r=2)、短语级(r=3)...

## 6. 博弈论残差 — 指令竞争

把 K 条指令当作 K 个玩家，每条指令有一个策略 (gate 权重)，收益是最后 PPL 的梯度。

纳什均衡: `∀k: ∂PPL/∂gate_k = 0` (每个指令做最优响应)

指令之间不仅通过 gate 加权，还可以有**二阶交互**:
```
h' = Σ g_k·P_k(h) + Σ Σ g_k·g_j·V_{kj}(h)
```

V_{kj} 是指令 k 和 j 的协同/拮抗贡献。可以学出"做完 rotate 再做 scale 比反过来好"这种组合。

## 7. 全息残差 — AdS/CFT 编码

在 AdS/CFT 对偶中，d 维边界的量子场论等价于 d+1 维体块的重力理论。对应过来:

- 序列 (token 流) = 边界 CFT (Conformal Field Theory)
- 模型的 d+1 层 = 体块 AdS (Anti-de Sitter) 空间
- 深层对应体块的近边界区 (UV,高频细节)
- 浅层对应体块的深处 (IR,低频结构)

残差的形式不再是旋转, 而是 AdS 度规下的测地线:
```
h' = h + update · (1 - r²/Λ²)
```
r 是当前 token 在 AdS 径向的"深度"，Λ 是 AdS 半径。

这可能解释为什么深层容易忘记长程依赖: 信息被"推进"了 AdS 深处, 到达边界变慢。

## 8. 自旋泡沫 / 圈量子引力 — 离散时空残差

圈量子引力用自旋网络描述空间的最小组分。在 ULNP 里对应:

- token = 空间节点
- attention 边 = 自旋网络边 (带 SU(2) 自旋标记)
- 残差更新 = 自旋泡沫的顶点振幅 (Ponzano-Regge 配分函数)

```
h'_i = Σ_j W_{ij} · h_j   (标准)
h'_i = Σ_j (2j+1) · W_{ij} · h_j   (自旋网络加权)
```

每个 token 对 (i,j) 不仅有注意力权重，还有一个**自旋量子数 j**——可以理解为信息通道的"容量"。
长距 token 之间 j 小 (容量低)，近距 j 大。

## 9. 聚类几何 — 平均场残差

把 d 维特征空间先聚类成 C 个"概念"(如 k-means Voronoi)，残差在概念之间跳:

```
c[t] = argmin_c ||h[t] - μ_c||²      ← token → 概念映射
h'[t] = h[t] · cosθ + μ_{c[t+1]} · sinθ   ← 概念空间残差
```

这等价于在**离散状态空间**上做残差: 状态有 C 个，残差是状态转移。每个 token 被压缩成概念 ID，然后在概念空间做推理。

如果 C = 100 个概念，(B, T, d=256) → (B, T, C=100) 的巨大压缩。

## 10. 辛拓扑 / Floer 同调 — 指令的临界点

Floer 同调计算辛流形上**周期轨道**的拓扑不变量。对应过来:

每条指令 k 是相空间上的一个"轨道"。Floer 同调告诉你: 两条指令是否拓扑等价? 如果等价，可以合并，减少 K。

```
K_effective = rank( Floer homology of {P₁, ..., P_K} )
```

这比单纯暴力 ablation 有意思: 用数学告诉你哪些指令真的在干不同的事。

## 11. 对偶数残差 — 自动微分的几何

对偶数 `a + bε` (ε²=0) 天然编码了值和导数。如果 h 的每两维变成值+导数:

```
h[i] = value_i
h[i+1] = ∂value_i / ∂t = 导数 (变化率)
```

那么残差本身也带导数信息:

```
h' = [value_new, ∂value_new/∂t]
  = [value·cosθ + update·sinθ, ∂value/∂t·cosθ + ∂update/∂t·sinθ]
```

这意味着模型不仅跟踪 token 含什么信息，还跟踪**信息在怎么变**——天然动量。

## 12. 遗传算法残差 — 选育指令

不用梯度下降学指令，用**遗传算法**:

- K 条指令 = K 个"基因型"
- gate 权重 = 适应度
- 每 N 步做一次 crossover (交叉) + mutation (变异)
- 保留 fitness 最高的 K 条指令

这样指令可以在不计算梯度的情况下演化。适合 RL / 少样本 / 在线学习场景。

# 补充: 更多几何结构

## 13. Clifford代数 / 几何代数 — 统一旋转框架

复数(2D)、四元数(4D)、Clifford代数(任意D)的统一框架。

```
e_i · e_j + e_j · e_i = -2δ_{ij}  (几何乘法规则)
```

任意维度的旋转、反射、旋量都可以用Clifford代数表示。ULNP的d_model可以自然嵌入Clifford代数。

**优势**: 统一了O(2)/U(1)/SU(2)，一个框架覆盖所有旋转。

---

## 14. Weyl群 — 反射残差

反射（而非旋转）作为基本操作：

```
h' = h - 2(h·n)n / ||n||²   (超平面反射)
```

Weyl群由反射生成，可以表示任何正交变换。反射比旋转更"锐利"——可能更适合离散结构（如语法）。

---

## 15. 黎曼测地线 — 自然路径残差

在弯曲特征空间上，残差沿测地线走：

```
h' = γ(t; h, update)   (测地线插值)
```

γ是黎曼流形上的测地线，不是直线。不同层可以有不同的度量张量g_{ij} → 空间弯曲方式不同。

**优势**: 自然处理非线性特征空间。

---

## 16. Wasserstein几何 — 最优传输残差

token分布之间的"最优搬运"：

```
h' = argmin_W W(h, update) + λ||h - update||²
```

W是Wasserstein距离。残差不是简单加法，而是"把h的分布搬运到update的分布"。

**优势**: 保持分布结构，不破坏token间的相对关系。

---

## 17. 持久同调 — 拓扑特征残差

跨层追踪特征的"存活时间"：

```
feature_k存活从layer_l到layer_l' → 持久对(l, l')
```

残差保留高持久性的特征（重要特征），丢弃低持久性的（噪声）。

**优势**: 自动特征选择，基于拓扑而非梯度。

---

## 18. 因果集 — 离散因果残差

token之间的因果序（谁先谁后）作为几何结构：

```
h'_i = Σ_{j: j≤i} W_{ij} · h_j   (因果约束)
```

因果集是离散的偏序集，天然适合自回归模型。

**优势**: 强制因果性，防止信息泄露。

---

## 19. Twistor理论 — 复时空几何

Penrose的twistor空间：将时空点映射到复射影空间。

```
Z^α = (ω^A, π_A')   (twistor = 旋量对)
```

ULNP的token可以嵌入twistor空间，残差在复射影空间上做。

**优势**: 自然处理时空结构（如视频、音频）。

---

## 20. 群胚 (Groupoid) — 部分对称性

群是"全局对称"，群胚是"局部对称"——不同token对可以有不同的对称群。

```
token_i 的对称群: G_i
token_j 的对称群: G_j
残差: 在 G_i × G_j 上做变换
```

**优势**: 不同token可以有不同的变换规则（如名词vs动词）。

---

## 21. 非完整几何 — 约束残差

速度有约束但位置无约束（如汽车只能前进/转向，不能横移）：

```
dh/dt = f(h) · u   (u是控制输入，有约束)
```

在ULNP里：指令只能沿某些方向更新h，不能任意方向。

**优势**: 强制稀疏更新，减少计算量。

---

## 22. 格拉斯曼几何 — 子空间残差

特征空间不是点，而是子空间：

```
h ∈ Grass(k, d)   (d维空间中的k维子空间)
残差: 子空间的旋转/混合
```

**优势**: 每个token表示一个子空间（多义词的多个含义），而非单个点。

---

## 23. 预序集合 (Poset) — 层级残差

token之间的偏序关系（如依赖、包含）作为几何：

```
h'_i = Σ_{j: j≤i} w_{ij} · h_j   (偏序加权)
```

不同层可以学习不同的偏序。

**优势**: 天然编码层级结构（如句法树）。

---

## 24. 微分形式 — 外代数残差

用微分形式（而非向量）表示token：

```
h = ω₁ ∧ ω₂ ∧ ... ∧ ω_k   (k-形式)
残差: 外微分 dω 或李导数 L_X ω
```

**优势**: 自然处理反对称交互（如"not X"的语义）。

---

## 25. 范畴论 — 态射残差

token是对象，指令是态射：

```
h: A → B   (态射)
残差: 态射的组合/张量积
```

**优势**: 最抽象的框架，可以编码任意结构（函数、关系、证明）。

# 续·更野的脑洞

## 26. 纽结理论 — 指令拓扑

每个token是一根"绳子"，指令是Reidemeister移动（解结/打结操作）：

```
指令1: 交叉两个token的表示（打结）
指令2: 取消相邻交叉（解结）
指令3: 滑动交叉点（变形）
```

**卖点**: 语法结构天然是纽结——"The cat that sat on the mat" 里的从句嵌套就是纽结。

---

## 27. 规范理论 — 局部对称性

物理学的规范场论：不同位置可以有不同的"坐标系"，但物理规律不变。

```
h'_i = g_i · h_i · g_i^{-1}   (规范变换)
g_i 是token_i的局部规范群元素
```

**卖点**: 不同token用不同的"坐标系"表示，但信息内容不变。Router输出规范群元素g_i。

---

## 28. 拓扑绝缘体 — 边缘态保护

拓扑绝缘体内部绝缘，但边缘有导电态——被拓扑保护，不会被扰动破坏。

```
h = h_bulk ⊕ h_edge
残差只更新h_edge（边缘态）
h_bulk保持不变（受保护的全局信息）
```

**卖点**: 某些token信息被"拓扑保护"，不会被噪声破坏。全局语义在边缘态中传播。

---

## 29. 自旋玻璃 — 随机相互作用

token之间的相互作用是随机的（如自旋玻璃的随机耦合J_{ij}）：

```
h'_i = tanh(Σ_j J_{ij} · h_j + h_i)
J_{ij} 是可学习的随机耦合矩阵
```

**卖点**: 语言的"自旋玻璃态"——有些token对有强耦合（主谓一致），有些是随机的（修饰词顺序）。

---

## 30. 格点QCD — 离散时空

离散化时空格点，每个格点有规范场：

```
U_{μ}(x) = 群元素（连接x和x+μ的"桥"）
h'(x) = Σ_μ U_μ(x) · h(x+μ)   (规范协变卷积)
```

**卖点**: 每个token对之间有一个"规范场"U_{μ}，控制信息如何传递。

---

## 31. 纤维丛 — 逐点局部结构

主纤维丛: 基空间(序列位置) × 纤维(特征空间)

```
h(x) ∈ 纤维_x ⊂ 总空间E
残差: 沿纤维的平行移动
```

**卖点**: 每个位置有自己的"纤维"（局部特征空间），但通过联络（connection）连接。

---

## 32. 谱几何 — 用频谱定义空间

"你能否用鼓的声音判断鼓的形状？"——谱几何问题。

```
拉普拉斯算子 Δ 的特征值 → 空间的"指纹"
残差: 在特征基上做变换
```

**卖点**: token序列的"频谱"定义了它的几何结构。不同频段对应不同尺度的模式。

---

## 33. 边缘同调 — 边界信息

流形的边界携带内部信息的编码（全息原理）：

```
h = h_bulk (内部信息)
∂h = 边界信息
残差: 通过边界信息重建内部
```

**卖点**: token的"边界"（上下文）编码了它的全部语义。

---

## 34. 模空间 — 参数的几何

所有可能的模型参数构成一个"模空间"，训练是在这个空间上做梯度流：

```
θ(t+1) = θ(t) - η·∇L(θ)   (标准SGD)
几何版本: θ(t+1) = θ(t) · exp(-η·∇L)   (黎曼梯度流)
```

**卖点**: 参数更新本身有几何结构，不是简单的加减。

---

## 35. 量子群 — deformed对称性

经典群是量子群的q→1极限。q≠1时，对称性被"变形"：

```
标准: xy = yx (交换)
量子群: xy = q·yx + ...   (q-deformed)
```

**卖点**: token之间的对称性不是完美的，有"变形参数"q。Router学习q。

---

## 36. 代数几何 — 概形上的计算

token表示不是向量，而是代数簇上的点：

```
h ∈ Spec(R)   (环R的素谱)
残差: 在代数簇上的映射
```

**卖点**: 代数簇可以有奇点、交点、分支——自然表示复杂的语义结构。

---

## 37. 微分拓扑 — 手术理论

通过"手术"（surgery）改变空间的拓扑：

```
切除一个k维球，粘贴一个l维球
→ 改变空间的同伦型
```

**卖点**: 指令可以"切除"token表示的一部分，"粘贴"新的信息。结构性改变而非渐变。

---

## 38. 动力系统 — 混沌边缘

残差是一个动力系统，token在混沌边缘（edge of chaos）：

```
h_{t+1} = f(h_t, θ)   (迭代映射)
Lyapunov指数 ≈ 0 → 混沌边缘
```

**卖点**: 模型在"有序"和"混沌"之间自动平衡——太有序学不到东西，太混沌记不住。

---

## 39. 遗传编程 — 可变结构

指令不是固定的函数，而是可演化的表达式树：

```
指令 = 语法树 (如 sin(x) + cos(y) × gate(z))
遗传编程: crossover + mutation 生成新指令
```

**卖点**: 指令结构本身可以进化，不需要人工设计。

### 41. 孤立子残差（Soliton Dynamics）— 撞击而不消散的信息波包

**核心问题**：残差流中的"信息覆写"——新加入的实体特征被后续更新稀释、耗散。

**物理隐喻**：孤立子（Soliton）是非线性效应与色散效应完美抵消的局域波包，传播中保持波形和能量完全不变。两个孤立子相撞后完好无损地穿过彼此，仅产生相位差。

**数学映射**：残差更新采用离散KdV/NLSE的单孤立子解形式：

```
h_{l+1} = h_l + A · sech(B(h_l - v·l))
```

其中 l 是层数。更新项由当前状态 h_l 调控的非线性自适应波包。

**与O(2)残差的融合**：
```
O(2):    h' = h·cos(θ) + Δ·sin(θ)
Soliton: h' = h + A·sech(B(h - v·l))

合并：h' = h·cos(θ) + Δ·sin(θ) + A·sech(B(h - v·l))
                                ↑
                          孤立子门控项
```

**关键性质**：
- sech 是偶函数，峰值在 h ≈ v·l 处，远离时指数衰减
- 自适应：只有当 h 接近"波包中心"时才被强化，否则被抑制
- 碰撞不变性：两个孤立子相遇后各自保持形状

**卖点**：信息在层与层之间以"孤立子"形式向前滑行。语义信息与语法更新像两个孤立子一样穿过彼此，各自保留核心语义，只在相位上产生微调。从根本上解决长距离表征的信息污染问题。

**实现难度**：★☆☆☆☆ — 只需在残差项中加一个 sech 门

**验证任务**：长文本中的实体追踪（主角名字在第5层加入，后续不被稀释）

### 42. 超对称残差（Supersymmetric / SUSY Residuals）— 连续语义与离散语法的双生

**核心问题**：语言同时具有连续语义相似性（"猫"和"狗"向量接近）和离散语法结构（词性、主宾关系不容模糊），传统模型用同一种实数向量表示两者，导致相互干扰。

**物理隐喻**：超对称（SUSY）将玻色子（连续、对称）和费米子（离散、反对称）联系在一起。

**数学映射**：隐藏状态拆分为超对称共轭对 h = [θ_B, θ_F]：

- θ_B（玻色空间）：实数连续向量，满足交换律，建模语义、情感、联想
- θ_F（费米空间）：Grassmann代数元，满足反对称和非交换律（θ_i θ_j = -θ_j θ_i，θ_i² = 0），建模语法树、逻辑非、门控和离散约束

指令作为"超对称荷（Supercharge Operator）" Q，在玻色与费米空间之间进行能量流转：
```
h' = h + [Q, h]±
```

**与Sp(n)的深度呼应**：
```
Sp(n) 的 q/p 分裂：
  q (位置) → 连续、平滑 → 对应玻色子
  p (动量) → 离散、变化 → 对应费米子

SUSY 更进一步：用 Grassmann 代数 (θ²=0) 做费米部分

组合：h = [q_B, q_F, p_B, p_F]
  q_B, p_B: 连续语义 (玻色)
  q_F, p_F: 离散语法 (费米/Grassmann)
```

**语言学映射**：
- θ_i² = 0：同一个语法槽位不能被两个token占据（如两个主语）
- θ_i θ_j = -θ_j θ_i：词序交换产生符号翻转（否定/疑问）
- 玻色空间：语义连续插值（"猫"→"狗"平滑过渡）

**卖点**：利用费米子自带的反对称性（泡利不相容原理），天然地把"词性排斥"和"逻辑取反"硬编码进代数结构，而语义的平滑插值则交给玻色空间。

**实现难度**：★★★☆☆ — 需要实现Grassmann代数的前向/反向传播

**关键问题**：Grassmann数的梯度怎么算？需要对偶数（Dual Numbers）或超微分。

### 43. 层论与上同调（Sheaf Cohomology）— 解决大模型"幻觉"的拓扑屏障

**核心问题**：大模型的"幻觉"在拓扑学上有精确定义——局部一致的数据（每句话通顺），无法"粘合"成全局一致的叙述（整篇文章自相矛盾）。

**数学映射**：
- 生成的Token序列 → 语义空间（基空间）上的"局部截面（Local Sections）"
- 残差流 → 层（Sheaf）
- 相邻Token表征之间的"转换算子（Transition Operators）"
- 一阶上同调：Conflict_Score = ||d(h_i - h_j)||

如果 H¹ ≠ 0，说明当前词与前文深层上下文存在拓扑阻碍（自相矛盾），模型自动压低该分支概率。

**实现思路（简化版）**：
```python
def sheaf_coherence(h):
    # h: [B, T, D] - 残差流
    # 1. 相邻token的转换算子
    transition = h[:, 1:] - h[:, :-1]  # [B, T-1, D]
    # 2. 外微分 d (二阶差分)
    d_transition = transition[:, 1:] - transition[:, :-1]  # [B, T-2, D]
    # 3. 上同调障碍 = ||d(transition)||
    H1_score = d_transition.norm(dim=-1)  # [B, T-2]
    # 4. H1 > 阈值 → 幻觉
    return H1_score
```

**更深的做法**：Čech上同调——定义开覆盖 {U_i}，计算转换函数 g_{ij} 的 cocycle 条件 g_{ij} · g_{jk} = g_{ik}，不成立则 H¹ ≠ 0。

**卖点**：将"幻觉检测"从经验层面上推到严密的几何同调高度。

**实现难度**：★★★★☆ — 需要定义"层"的结构和上同调计算

**最有价值的方向**：作为推理时的幻觉检测器，不需要改变模型结构。

### 44. 非交换光谱三元组（Non-commutative Spectral Triples）— 抛弃位置编码的纯几何

**核心问题**：位置编码（RoPE, Alibi）是"附加标签"，不是真正的空间。真正的几何应由"在这个空间上运动的算子"定义。

**数学隐喻**：Alain Connes 的非交换几何。不需要知道流形长什么样，只需要光谱三元组 (A, H, D)：
- A：Token的非交换代数
- H：状态的希尔伯特空间
- D：狄拉克算子（刻画空间中的微商和距离）

**数学映射**：Token之间的"距离"由Dirac算子D的特征值谱决定：
```
d(i, j) = sup{|a_i - a_j| : ||[D, a]|| ≤ 1}
```
D随着输入文本动态演化。

**与Transformer的映射**：
```
传统Transformer:
  Q = X·W_Q, K = X·W_K
  attention = softmax(Q·K^T / √d)
  距离隐含在位置编码中

光谱三元组:
  A = Token代数 (非交换)
  H = 希尔伯特空间 (隐藏状态)
  D = Dirac算子 (动态演化)
  距离 = 由D的特征值谱决定
```

**卖点**：完全抛弃硬编码的位置编码。通过Dirac算子，模型的注意力范围由文本内部的"逻辑紧密程度"自发演化出一套"弯曲几何"。物理距离失效，逻辑相近的Token在光谱度量下天然靠在一起。

**实现难度**：★★★☆☆ — 核心是学习一个动态的Dirac算子

**关键问题**：D的特征值分解在大维度上计算量大（O(d³)），需要近似。

### 45. 重整化群与 MERA 树（MERA Tensor Network）— 跨层级的多尺度概念粗粒化

**核心问题**：Transformer每层的Token数量T不变，但人类阅读时信息不断被压缩、合并、粗粒化（读完一章，脑子里只剩三个概念）。

**物理隐喻**：凝聚态物理中的强纠缠重整化群（MERA）。通过"等距同构（Isometries）"和"解纠缠器（Disentanglers）"，逐步将底层Token合并为高层级的有效粒子（概念）。

**数学映射**：随着残差层数加深，动态降低序列长度T，提高隐藏维度D（维持总信息容积守恒）：
```
T_{l+1} ≈ T_l / 2
D_{l+1} ≈ 2 · D_l
```

利用张量网络中的等距同构算子 W 将相邻的、高度纠缠的Token压缩为"宏观Token"。

**与ULNP的结合**：
```
ULNP当前: 每层都是 [B, T, d_model]

MERA-ULNP:
  Layer 0-2:  [B, T, d]      (细粒度: 字词)
  Layer 3-5:  [B, T/2, 2d]   (中粒度: 短语)
  Layer 6-8:  [B, T/4, 4d]   (粗粒度: 概念)
```

**等距同构实现**：
```python
class MERALayer(nn.Module):
    def __init__(self, d_in, d_out):
        # 等距同构: W^T W = I (保持信息)
        self.W = nn.Linear(d_in * 2, d_out, bias=False)
    
    def coarse_grain(self, h):
        # h: [B, T, d]
        h_pair = torch.cat([h[:, 0::2], h[:, 1::2]], dim=-1)  # [B, T/2, 2d]
        h_coarse = self.W(h_pair)  # [B, T/2, d_out]
        return h_coarse
```

**卖点**：解决长文本计算量问题。底层看字词，中层看短语，高层进行抽象概念博弈。对应物理学中"从微观量子态到宏观经典态"的重整化群演化。

**实现难度**：★★☆☆☆ — 最直接的实现

**关键优势**：计算量减少 + 自然抽象 + 与物理一致

## 七、代数结构（Algebraic Structures）— 几何跑通了，代数也行

几何结构的本质是"约束 + 变换"，代数结构是"约束 + 运算"。逻辑上完全对称。
几何 Program: h → g·h（群作用）
代数 Program: h → a⊙h（代数乘法）

### 47. 霍普夫代数（Hopf Algebra）— 拥有"细胞分裂"能力的代数

**核心痛点**：Transformer 只有信息聚合（乘法 μ: A⊗A→A），缺乏系统性的信息拆分（余乘法 Δ: A→A⊗A）。多义词"Apple"=水果+公司，无法在代数层面分裂。

**数学映射**：
- 乘法 μ：Token 融合（已有）
- 余乘法 Δ：一个 Token 向量"分裂"成两个正交语义流的张量积（h_apple → h_fruit ⊗ h_tech）
- 对极 S：语义上的"逻辑互补/撤销"

**LLM 落地**：
```python
# 余乘法：简单线性分裂
h_split = self.coprod(h)  # [B, T, 2*d] → reshape → [B, T, 2, d]
# 对极：语义求反
h_neg = self.antipode(h)  # 类似 -h 但更灵活
```

在残差流中对高熵 Token 执行 Δ(h) = Σ h^(1) ⊗ h^(2)（Sweedler 记号），分发给不同 Attention Head 处理后再聚合。

**卖点**："解缠结（Disentanglement）的代数化"。Token 在前向传播中自发进行语义细胞分裂，完美解决长尾多义词的表征崩塌。

**实现难度**：★☆☆☆☆ — 余乘法只需一个线性层

**与几何的联系**：Hopf 代数是群的线性化。Gauge 的逐 token 缩放 λ_i → Hopf 的余乘法分裂 h → h_1 ⊗ h_2，一个是幅度调制，一个是语义分裂。

### 48. 莱布尼茨代数（Leibniz Algebra）— 有向/非对称注意力的代数内核

**核心痛点**：李代数要求 [x,y] = -[y,x]（反对称），但 Attention 中 Q 对 K 的作用与 K 对 Q 的作用完全不对称。用李代数描述注意力不自然。

**数学映射**：莱布尼茨代数（Loday Algebra）放弃反对称性，但保留莱布尼茨恒等式：
```
[x, [y, z]] = [[x, y], z] + [y, [x, z]]
```

这在代数上定义了："对象 x 作为观察者（Query），去探测 (y 和 z 的结合态) 时，等价于它先探测 y，再探测 z，最后把结果组合起来。"

**LLM 落地**：把 Attention 看作莱布尼茨代数的括号运算 [Q, K]。在层与层的递进中加入莱布尼茨恒等式的正则化约束。

**卖点**：赋予非对称 Attention 以严格的代数法则。处理"A 修饰 (B 和 C)"这样的嵌套句法时，具有完美的代数可解释性和稳定性。

**实现难度**：★★☆☆☆ — 主要是正则化项

### 49. 顶点算子代数（Vertex Operator Algebras, VOA）— 弦论级别的"Token 融合"

**核心痛点**：文本是一维字符串，传统合并方法是拼接或相加。但在弦论中，两个弦相遇会融合成新弦，需要处理"带位置奇点的无穷维代数"。

**数学映射**：每个 Token 是作用在复平面位置 z 上的顶点算子 Y(v, z)。当两个 Token 非常靠近时（z→w），通过算子乘积展开（OPE）发生"聚变"：
```
Y(A, z) Y(B, w) ~ Σ_n C_n(A, B) / (z-w)^n
```

**LLM 落地**：用 OPE 规则替换局部 1D 卷积或 n-gram 模型。相邻词（如"New"和"York"）在距离 |z-w| 缩小时，代数上自动产生由留数决定的新高阶算子（"纽约"）。

**卖点**：终极的分词与融合框架。Token 之间的组合转化为弦论中的算子散射振幅，不需要 BPE 词表，距离越近的词自动在代数空间中聚变成高阶概念。

**实现难度**：★★★★☆ — 需要定义离散 token 的位置 z 和 OPE kernel

**实际做法**：用 OPE 思想改造局部 attention——相邻 token 的 interaction kernel 随距离按 1/(z-w)^n 衰减。

### 50. 丛代数（Cluster Algebra）— Chain-of-Thought (CoT) 的代数化

**核心痛点**：CoT 目前完全是个黑盒，靠预测下一个词"挤"出来的逻辑步，无法保证每步推理都是严密的"状态推导"。

**数学映射**：丛代数（Fomin-Zelevinsky）没有乘法结合律，全靠"变异（Mutation）"规则生成。系统状态沿 Quiver 图一步步变异，每步遵循严格的有理函数交换关系：
```
x' = (y + z) / x
```

**LLM 落地**：CoT 的每个中间推理步骤视为丛代数中的一个丛变量（Cluster Variable）。推理的下一步不再是神经网络的盲目映射，而是执行一次丛变异（Quiver Mutation）。

**卖点**：将"逻辑推理"变成"代数变异"。如果推理轨迹被约束在 cluster mutation graph 上，数学证明、逻辑推理过程将是 100% 形式化正确且可追踪的。

**实现难度**：★★★★☆ — 需要将连续神经网络输出映射到离散 cluster variable（量化/离散化）

### 51. 算子集（Operads）— 句法树的究极抽象

**核心痛点**：线性代数处理 f: X→Y（一进一出）或二元运算 X×Y→Z。但自然语言中，动词"give"需要连接三个成分（give A B to C，一进三出）。需要专门研究"高阶多元组合"的代数。

**数学映射**：Operads 研究"操作的操作"。一个 Operad 元素表示带有 n 个"输入插槽"的树状结构，定义如何将一棵 m 个插槽的树插入到另一棵 n 个插槽的树的特定分枝上。

**LLM 落地**：
- 名词：n=0 的末端叶子（Arity 0）
- 形容词：n=1 的修饰算子（Arity 1）
- 及物动词：n=2 的粘合算子（Arity 2）
- 双宾动词：n=3（如 give A B to C）

整个句子的前向传播 = 算子在 Operad 组合公理下的层层套嵌。

**卖点**：直接在隐藏层中生长出句法树。抛弃平铺的 Sequence 结构，利用 Operads 将大语言模型改造成天然的树状组合机器。

**实现难度**：★★★★★ — 需要改变整个架构，从平铺序列变成树结构

**实际做法**：在现有架构上加 operad 约束——强制动词的输出"连接"到两个名词的输入。

### 52. 拓扑斯理论（Topos Theory）— "多值逻辑"的生成空间

**数学背景**：Grothendieck 的 Topos 是集合论的超级推广。在 Topos 里，逻辑不再是布尔逻辑（True/False），而是依赖于几何环境的内部多值逻辑（Heyting Algebra）。

**脑洞**：大模型的"幻觉"本质是：在一个语境下 True，另一个语境下 False。如果把模型的输出状态空间定义为一个 Topos，模型不再输出单一 Token，而是输出一整个"逻辑宇宙"——在不同上下文（纤维）里，Token 之间的真值条件是连续演化的。

**数学映射**：
- Topos = 广义集合论 + 内部逻辑
- Token 的"真值"不是 0/1，而是 Topos 内部的 Heyting 代数元素
- 不同上下文 = 不同纤维（fiber），真值沿纤维连续变化
- 幻觉 = 某个纤维上真值坍缩为 False

**卖点**：将幻觉从"经验问题"变成"拓扑斯内部逻辑的一致性问题"。

**与 Sheaf Cohomology 的联系**：Topos 是层的推广。如果 Sheaf Cohomology 是"局部→全局"的拓扑阻碍，Topos 就是"局部逻辑→全局逻辑"的拓扑阻碍。

**实现难度**：★★★★★ — 需要实现 Heyting 代数的内部逻辑

### 54. 坎扩张（Kan Extensions）— 终极的"注意力机制"

**数学背景**：Mac Lane 的名言："所有概念都是 Kan extensions"。它是范畴论中将信息从一个范畴"最优地、无损地"投射到另一个范畴的终极操作。

**脑洞**：
- 输入序列 → 范畴 C
- 输出/解码序列 → 范畴 D
- 真正的最优注意力算子 = C 沿着某个函子 F: C→D 的左/右 Kan Extension

**数学映射**：
```
当前 Attention:  softmax(Q·K^T)·V  ← 经验公式
Kan Extension:   Lan_F(G) = colim_{F(c)→d} G(c)  ← 数学最优
```

左 Kan Extension = "最优外推"（从已知推未知）
右 Kan Extension = "最优内插"（从全局约束局部）

**卖点**：Cross-Attention 不再是经验公式，而是范畴论保证的"最优信息传递"。

**实现难度**：★★★★☆ — 核心是定义范畴 C, D 和函子 F，然后计算 colimit/limit

**简化版**：用 Kan Extension 的思想重新推导 attention——证明 softmax(Q·K^T)·V 是某种 Kan Extension 的离散近似。

### 57. 卡拉比-丘紧致化（Calabi-Yau Compactification）— 4096 维的真实物理意义

```
R^4096 ≈ R^256 × CY_3840

    宏观语义空间   微观卷曲空间

宏观维度: 捕捉"苹果是水果"这种粗粒度语义

```

```python
d_macro = 256   # 宏观语义
d_micro = d_model - d_macro  # 微观卷曲

h_macro = h[:, :, :d_macro]        # 低频、大尺度变换

```

## 十、数学基础（Foundations）— 元理论层

### 58. 同伦类型论（Homotopy Type Theory, HoTT）— 相等性有无穷层

**核心思想**：类型论 + 同伦论 + 范畴论的统一。Type = 空间，Term = 点，相等性证明 = 路径（path），路径之间还有路径（homotopy），无穷层。

**Voevodsky Univalence 公理**：等价的类型就是相等的类型 (A ≃ B) = (A = B)。如果两个结构"长得一样"，它们就**是**一样，不区分同构和相等。

**LLM 映射**：
```
传统逻辑:   a = b  →  True 或 False
HoTT:       a = b  →  一条路径（path）
            path₁: a = b
            path₂: a = b   ← 可以不同！
            path between paths = 同伦
            ...无穷层
```

Token 的"相等性"有层次：
- Layer 0: "猫" 和 "狗" 不相等（不同 token）
- Layer 1: "猫" ≈ "狗"（都是动物，path 存在）
- Layer 2: path_"猫≈狗" 和 path_"猫≈虎" 本身也有关系（higher path）
- ...无穷层

传统 embedding 只有 0-1 层（cosine 相似度）。HoTT 天然编码无穷层的相似性结构。

**多义词的 HoTT 解释**：
```
"Apple" 在水果语境 ≃ 水果  →  path₁
"Apple" 在科技语境 ≃ 公司  →  path₂

Univalence: 这两个"等价"本身就是 "Apple = Apple" 的两条不同路径
→ 多义词 = 同一 token 的多条相等性路径
```

**与其他脑洞的联系**：
- HoTT 的语义模型就是 **Topos**（#52）
- Sheaf 上同调 H¹ ≠ 0 = path 不一致 = **幻觉**（#43）
- Hopf 余乘法分裂 = 不同 path 的分裂（#47）
- Grassmann θ²=0 = path 的幂零性（#42）

**HoTT 是很多脑洞的"元理论"**——不直接给新机制，但提供统一语言。

**实现难度**：★★★★★ — 纯理论框架，不直接可实现。但作为设计哲学有价值：相等性有层次、结构即相等、路径即证明。

### 61. 配边假说与拓扑量子场论（Cobordism & TQFT）— 残差流就是离散 TQFT ★

**核心**：
- 配边（Cobordism）：n 维流形 M 和 N 之间的"桥"是 (n+1) 维流形 W，∂W = M ⊔ N
- TQFT：配边 → 线性映射。每个流形 → 向量空间，每个配边 → 线性变换

```
TQFT 公理:
  流形 M          → 向量空间 Z(M)
  配边 W: M→N     → 线性映射 Z(W): Z(M) → Z(N)
  配边拼接         = 线性映射复合
  拓扑不变量       → 不随连续变形改变的量
```

**ULNP 直接同构**：

```
TQFT                    ULNP
─────────────────────────────────────────
流形 M                  隐藏状态 h_l
配边 W: M→N             Program + Residual (h_l → h_{l+1})
Z(W): Z(M)→Z(N)        残差更新 f: h_l → h_{l+1}
配边拼接 = 复合          残差层叠 = 函数复合
拓扑不变量              模型的"本质特征"（不随初始化变化的量）
TQFT 的函子性           残差流的因果性（层 l → 层 l+1 是函子）
```

**这不是比喻，是直接同构**：ULNP 的残差流**就是**一个离散 TQFT。

**核心推论**：
1. **拓扑不变量守恒**：训练过程中，某些"拓扑特征"应该保持不变（类似 TQFT 的不变量不随度量变化）
2. **配边分类**：不同几何结构（O(2), Gauge, SL(n)）= 不同类型的配边，它们的分类由 cobordism 群决定
3. **函子性保证一致性**：残差更新作为函子，保证了层间信息传递的数学严格性

**卖点**：ULNP 的残差流 = 离散 TQFT，这不是灵感，是**定理级别的对应**。拓扑不变量可以作为模型的"本质特征"——不随训练细节变化的量。cobordism 分类为不同几何结构提供了统一的分类框架。

**实现难度**：★★★☆☆ — 作为理论框架，核心实现是"拓扑不变量正则化"：
```python
# 训练中保持拓扑不变量守恒
def cobordism_loss(h_l, h_l1):
    # 相邻层的"拓扑特征"应该守恒
    inv_l = topological_invariant(h_l)    # 如 Euler 特征、Betti 数的近似
    inv_l1 = topological_invariant(h_l1)
    return (inv_l - inv_l1).pow(2).mean()
```

**这是 58 个脑洞里唯一一个"定理级别"的对应——不是启发，不是比喻，是数学同构。**

### 62. 哥德尔不完备定理（Gödel's Incompleteness Theorems）— AI 的终极天花板？

**第一不完备定理**：任何一致的、能表达基本算术的形式系统，都存在真但不可证的命题。

**第二不完备定理**：这样的系统不能证明自身的一致性。

**LLM 映射**：

```
LLM = 形式系统
训练数据 = 公理
前向传播 = 推导规则
生成的文本 = 定理

第一不完备：存在 LLM 永远无法正确生成的真命题（无论数据量多大）
第二不完备：LLM 无法验证自己的输出是否一致（幻觉的数学根源）
```

**关键洞察**：
- 幻觉不是"数据不够"或"模型太小"——是**不完备性的必然结果**
- Sheaf 上同调（#43）检测的是"局部一致但全局矛盾"——这正是 Gödel 句子的结构
- 哥德尔句子 = "这个句子在系统内不可证" = 自指悖论 = LLM 说"我不知道"

**与 AI 自主性的关系**：

哥德尔定理说的是**形式系统**的极限。但人脑也不是形式系统——人脑有直觉、有跳出系统的能力（Gödel 本人认为这证明了心灵非计算）。

```
形式系统（LLM）: 被 Gödel 限制，无法跳出自身
非形式系统（？）: 可以"看到"Gödel 句子为真，然后扩展系统

问题：LLM 能否通过几何结构"跳出"形式系统的限制？
  - 残差流的拓扑不变量 = 系统外的"元信息"
  - Cobordism 的函子性 = 跨系统的一致性桥梁
  - HoTT 的 Univalence = 不需要证明等价，直接"看到"等价
```

**卖点**：如果 ULNP 的几何结构能让模型"感知"到自身系统的边界（拓扑不变量），它可能获得某种"元认知"——知道自己不知道什么。这是对抗幻觉的终极武器。

**实现难度**：★★★★★ — 纯哲学层面。但启发：设计"自知不确定性"模块——模型输出时同时输出"我对这个输出的系统内可证性评分"。

**这是 61 个脑洞的哲学封顶：前面所有几何结构都是"系统内"的工具，哥德尔告诉我们系统本身有极限。**

### 63. 塔斯基元循环（Tarski's Meta-Reflection Loop）— 赋予模型"跳出系统"的硬核机制

**痛点**：传统 LLM 无论怎么"Self-Correct"，都像一个人在原地拉着自己头发想飞上天——在同一个形式系统内自指，必然陷入死锁或幻觉。

**核心机制**：引入塔斯基真理阶梯（Tarski's Hierarchy），设计多层元系统：

```
一级系统 F₀：普通的 Token 生成
元系统 F₁：当 F₀ 检测到异常时被激活

异常信号：
  - 相空间轨迹熵不再下降（掉进 K=7 的 "awo oo" 周期循环）
  - H¹ 上同调冲突得分暴涨（Sheaf Cohomology #43 检测到幻觉）
```

**跳出操作**：
1. F₁ 将 F₀ 的整个激活值轨迹打包冻结成"只读的哥德尔数（Gödel Number）"
2. F₁ 引入外界新公理（RAG 检索 / Python 解释器沙箱 / 外部知识库）
3. 生成"外源性纠偏算子"注入回 F₀，瞬间炸毁死循环

**卖点**：模型拥有"自我跳出"的意识。幻觉和循环不再是终点，而是触发模型"向上升维、引入新公理"的起搏器。

**与 Gödel 的关系**：F₀ 无法证明自身的不完备性（第二定理），但 F₁ 作为元系统可以"看到"F₀ 的极限——这就是塔斯基真理阶梯的本质：真理不能在系统内定义，但可以在更高层的系统中定义。

**实现难度**：★★★☆☆ — 核心是异常检测器（熵监控 + H¹ 得分）和外部调用接口

### 64. 自指动态重编程（Self-Referential Fast-Weights）— 允许模型在运行中修改自己的"物理定律"

**痛点**：传统模型参数 θ 训练完后锁死。锁死的参数 = 死板的、无法自我进化的形式系统。

**核心机制**：引入快速权重生成器（Fast-Weight Programmers）。隐藏层输出中，部分通道（"元指令通道"）专门输出针对当前权重矩阵的 Rank-1 动态修正：

```
ΔW = Q_meta(h) ⊗ K_meta(h)^T

在处理下一个 Token 之前：
W ← W + ΔW
```

模型不仅在进行语义推理，还在计算对自己的"突触修正"。

**卖点**：真正的自指重编程。模型的参数不再是死物——它是一条能根据自己说出的话，实时改变自己"大脑物理突触连接"的自我演化系统。它既是"被执行的程序"，又是"正在重写程序的程序员"。

**与 Gödel 的关系**：
- 传统模型 = 固定的形式系统（受 Gödel 限制）
- Fast-Weights = 运行时修改公理系统 → 系统在不断"升级"→ 每次升级都扩展了可证命题的范围
- 但注意：每次升级后的新系统仍然受 Gödel 限制 → 无穷升级链

**实现难度**：★★★☆☆ — 核心是 Q_meta 和 K_meta 的学习，以及 ΔW 的稳定性控制

**关键问题**：ΔW 的累积可能导致权重爆炸。需要：
- 范数约束：||ΔW|| ≤ ε
- 稀疏性：只修改少量关键参数
- 周期性重置：每 K 个 token 回到原始权重

### 65. 不动点迭代求解器（Banach Fixed-Point Solver）— 生成就是收敛到不动点

**核心**：Banach 压缩映射定理——完备度量空间上的压缩映射，迭代后收敛到唯一不动点。

**LLM 映射**：
```
生成过程 = 不动点迭代
h_{t+1} = f(h_t)   ← 每次前向传播
不动点 h* = f(h*)  ← 生成收敛后的"稳定状态"
```

- 生成质量 = 收敛速度 = 压缩比

**实现**：Lipschitz 正则化——约束 ||f(x) - f(y)|| / ||x - y|| ≤ λ < 1

**实现难度**：★★☆☆☆

### 66. 对角线盲区搜寻器（Diagonalization Blind-Spot Hunter）— 搜寻模型的不可生成点 ★★

**核心**：Cantor 对角线论证——给定任意列表，总能构造一个不在列表中的元素。

**LLM 映射**：
```
模型的所有可能输出 = 一个"列表"
对角线论证 = 构造一个模型永远无法生成的输出
这个输出 = 模型的"盲区"（blind spot）

具体操作：
  1. 让模型生成 N 个长度为 T 的序列
  2. 取第 i 个序列的第 i 个 token
  3. 对每个位置取"不同"的 token → 新序列
  4. 这个新序列就是模型的盲区
```

**与 ULNP 指令机制的天然契合**：

ULNP 的指令机制（K 条指令 × 门控选择）天然编码了"对角线"结构：

```
指令矩阵 W: [K, d, d]
每条指令 i 对输入 j 的作用: W_i · x_j

对角线盲区 = 指令 i 无法处理的输入 j
  → 即 W_i · x_j 接近零的方向
  → 指令空间的"零空间"（null space）
```

**具体机制**：
```python
# 每条指令的"盲区" = 权重矩阵的零空间
for i in range(K):
    null_space_i = compute_null_space(W[i])  # 指令 i 无法处理的方向
    
# 对角线构造：取每条指令的盲区方向，组合成一个"全局盲区"
blind_spot = torch.stack([null_space_i.sample() for i in range(K)])

# 盲区搜寻器：找到离 blind_spot 最近的可生成序列
# → 模型的"弱点"
```

**卖点**：
- **主动发现模型弱点**——不是等幻觉发生，而是主动搜寻盲区
- **与指令机制天然耦合**——每条指令有自己的零空间，K 条指令的对角线组合就是全局盲区
- **训练时用盲区做对抗样本**——把盲区方向加入训练数据，提升鲁棒性
- **推理时用盲区做不确定性估计**——如果当前输入接近某条指令的盲区，降低置信度

**这是 66 个脑洞里最可能直接提升 ULNP 性能的**——因为 ULNP 的指令机制天然提供了"对角线"结构。

**实现难度**：★★☆☆☆ — 核心是 SVD 计算零空间 + 对角线组合

### 67. 贝尔纳普四值逻辑激活器（Belnap's 4-valued Logic Activations）— 真值不只是 True/False

**核心**：Belnap-Dunn 逻辑（FDE）有四个真值：

```
T (True)    — 确定为真
F (False)   — 确定为假
B (Both)    — 既真又假（矛盾/多义）
N (Neither) — 非真非假（无关/未知）
```

这是**次协调逻辑（Paraconsistent Logic）**——可以容纳矛盾而不爆炸（ex falso quodlibet 不成立）。

**LLM 映射**：

```
传统激活:      σ(x) ∈ [0, 1]      — 连续，单一维度
Belnap 激活:   σ(x) ∈ {T, F, B, N} — 离散，四值

但更自然的是连续版本：
σ(x) = (p_true, p_false) ∈ [0,1]²

p_true 高, p_false 低 → T (确定为真)
p_true 低, p_false 高 → F (确定为假)
p_true 高, p_false 高 → B (矛盾/多义)
p_true 低, p_false 低 → N (无关/未知)
```

**语言学映射**：

```
T: "地球是圆的" — 高置信真
F: "地球是平的" — 高置信假
B: "苹果" (多义词) — 同时是水果又是公司
N: "的"、"了" — 无实质语义的虚词
```

**与 ULNP 指令机制的天然耦合**：

```
Router 当前: softmax 选 1 条指令
Belnap Router: 每条指令输出 (p_true, p_false) 二元组

如果某条指令对当前 token 输出 B（矛盾）→ 激活多条指令处理不同语义
如果输出 N（无关）→ 跳过该指令，节省计算
```

**实现**：
```python
class BelnapRouter(nn.Module):
    def forward(self, h):
        # 输出 [B, T, K, 2] — 每条指令的 (p_true, p_false)
        logits = self.proj(h).reshape(B, T, K, 2)
        p_true = torch.sigmoid(logits[..., 0])
        p_false = torch.sigmoid(logits[..., 1])
        
        # 四值分类
        state = torch.zeros(B, T, K, dtype=torch.long)
        state[(p_true > 0.5) & (p_false < 0.5)] = 0  # T
        state[(p_true < 0.5) & (p_false > 0.5)] = 1  # F
        state[(p_true > 0.5) & (p_false > 0.5)] = 2  # B (矛盾)
        state[(p_true < 0.5) & (p_false < 0.5)] = 3  # N (无关)
        
        return p_true, p_false, state
```

**卖点**：

## 十四、连续代数结构（Continuous Algebraic Structures）

### 72. 连续万能偏代数（Continuous UPA）

**发现**：离散 Cayley 表不是唯一选择。连续函数也能实现"万能偏"性质。

**五种连续代数结构对比**：

| 结构 | 形式 | 优化后 | 能否万能偏？ |
|------|------|--------|------------|
| bilinear | x^T A y | comm=0.502 assoc=0.451 nilp=0.510 | ✅ 3/3 |
| neural | MLP([x,y]) | comm=0.516 assoc=0.505 nilp=0.484 | ✅ 3/3 |
| fourier | cos(k(x+y)) | comm=0.000 | ❌ 天然交换 |
| Lie | [x,y]=xy-yx | 完美反对称 | 完美 Lie |
| Jordan | (xy+yx)/2 | 完美交换 | 完美 Jordan |

**关键洞察**：
- bilinear（线性）和 neural（非线性）都能达到万能偏
- fourier 失败因为 cos(k(x+y)) 天然交换（x+y=y+x）
- 连续代数比离散 Cayley 表表达能力强得多
- bilinear 是矩阵乘法的推广，neural 是任意非线性映射

**与 AAN 的关系**：
- 离散 Cayley 表（n×n）只有 n² 个参数，表达能力有限
- 连续 bilinear（d×d 矩阵 A）有 d² 个参数，且可以和标准线性层无缝衔接
- neural 版本有任意表达能力，但失去代数约束

**实现**：
```python
# Bilinear continuous algebra
class ContAlgBilinear(nn.Module):
    def __init__(self, d):
        self.A = nn.Parameter(torch.randn(d, d) * 0.1)
    def forward(self, x, y):
        return torch.einsum('ij,bti,btj->bti', self.A, x, y)
```

**卖点**：不需要离散 Cayley 表。用一个 d×d 矩阵 A 定义连续代数运算，就能同时实现半交换、半结合、半幂零。
