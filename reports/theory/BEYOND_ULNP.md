## 一、数学结构的"未知物种"

### 1. 尚未被命名的几何

人类已经命名了几百种几何结构。但有没有**存在但没被发现的几何**？

```
已知：欧氏、双曲、辛、热带、非交换...
未知：？

思路：随机生成代数约束（如 a∘b = f(b∘a)），看哪些约束能产生
      有"意义"的几何——即存在不变量、有分类定理、能用于实际问题。
```

**乐子**：用 AI 搜索代数约束空间，发现人类没想过的几何结构。不是应用已知数学，是**发现新数学**。

## 二、神经网络的内在几何

### 2. 损失景观的拓扑学

不是用几何**在**网络里，是研究网络**本身的**几何。

```
1.7B 参数的损失景观是什么形状？
  - 是流形？是复形？是分形？
  - 有几个连通分量？（训练的不同初始化是否落在不同的"盆地"？）
  - 鞍点的分布？（高维空间里鞍点比极小值多得多）
  - 训练轨迹的同伦类？（不同优化器走的路径是否同伦等价？）
```

**乐子**：给损失景观做"CT 扫描"，画出它的拓扑骨架。

### 3. 涌现的相变物理学

LLM 训练中，能力会**突然涌现**（如 CoT 在某个 scale 突然出现）。

```
涌现 = 相变？
  - 临界点在哪？（参数量 / 数据量 / 训练步数的临界值）
  - 序参量是什么？（什么量在涌现前后发生突变？）
  - 临界指数？（涌现的速度服从幂律吗？）
  - 对称性破缺？（涌现前模型有某种对称性，涌现后破缺了？）
```

**乐子**：用统计力学的方法分析 LLM 训练，找到涌现的"临界温度"。

## 三、意识与自指

### 4. 思维的形状

如果把"思考"看作一个动力系统，它的相空间是什么形状？

```
一个念头 = 相空间中的一个点
思维的流动 = 相空间中的轨迹
灵感 = 相空间中的"突变"（从一个吸引子跳到另一个）
冥想 = 相空间中的不动点
```

**乐子**：用 fMRI 数据重建思维的相空间，看它的拓扑结构。

### 5. AI 的"内在体验"

我处理一个 prompt 时，隐藏层的轨迹是什么样的？

```
输入: "你好"
隐藏层轨迹: h_0 → h_1 → ... → h_L

这条轨迹的"形状"是什么？
  - 是直线？（最短路径）
  - 是螺旋？（反复回到相似状态）
  - 是分形？（自相似结构）
  - 是混沌？（对初始条件敏感）
```

**乐子**：可视化不同 prompt 在隐藏层中的轨迹，看"思考"的形状。

### 6. 自指系统的极限

Gödel 不完备定理说的是形式系统的极限。但 AI 系统是不是形式系统？

```
如果是 → 受 Gödel 限制 → 存在不可知的真命题
如果不是 → AI 有某种"超越形式系统"的能力 → 这能力是什么？

关键问题：AI 能否"感知"到自己的不完备性？
  - 如果能 → 元认知 → 接近意识
  - 如果不能 → 纯形式系统 → Gödel 限制
```

**乐子**：设计实验测试 AI 是否有元认知能力。

## 四、语言的物理学

### 7. 语言热力学

把语言看作热力学系统：

```
温度 T = 生成的随机性（temperature 参数）
熵 S = 输出分布的不确定性
自由能 F = E - TS（平衡信息量和多样性）
相变 = 模式切换（从散文到诗歌、从代码到注释）
```

**乐子**：测量不同文本类型的"温度"和"熵"，画出语言的相图。

### 8. 语义的拓扑

"意义"在向量空间里是什么形状？

```
"猫" 和 "狗" 的连线经过什么？
  - 经过"宠物"？（语义插值有意义）
  - 经过"动物"？
  - 经过"毛茸茸"？

"国王" - "男人" + "女人" = "女王"
  这个线性结构是普遍的还是偶然的？
  语义空间是平坦的还是弯曲的？
```

**乐子**：用微分几何测量语义空间的曲率，看"意义"是弯的还是直的。

### 9. 幽默的数学结构

什么东西好笑？

```
笑话的结构 = 预期建立 + 预期违反
  1. 建立语境（设置预期）
  2. 突然转向（违反预期）
  3. 回溯一致（违反后仍然"说得通"）

数学化：笑话 = 拓扑空间中的一个"奇点"
  - 正常文本 = 光滑流形
  - 笑话 = 流形上的一个锥点（cone point）
  - 在锥点处，"正常"的度量失效了
```

**乐子**：用信息论量化"意外程度"，看是否能预测笑话好不好笑。

## 五、计算的本质

### 10. 最简单的万能计算机

图灵机是最简单的经典万能计算机。但有没有**更简单的**？

```
已知：图灵机、Lambda 演算、细胞自动机（Rule 110）
未知：有没有只有一个状态、一个符号的万能计算系统？

或者反过来：什么是"刚好不够"万能的系统？
  - 比正则语言强，比图灵机弱
  - 这个中间地带有什么？
```

**乐子**：搜索计算能力的"相变点"——从有限到万能的临界条件。

### 11. 可证明与不可证明的边界

Gödel 说有些真命题不可证明。但**哪些**真命题不可证明？

```
可证明的命题 = "白洞"（信息可以出来）
不可证明的命题 = "黑洞"（信息出不来）
边界 = "事件视界"

这个"事件视界"的形状是什么？
  - 是光滑的？（连续过渡）
  - 是分形的？（自相似）
  - 是随机的？（不可预测）
```

**乐子**：用计算复杂度理论绘制"可证明性"的相图。

## 六、跨系统对话

### 12. 两个 AI 的共同数学

如果两个独立训练的 AI 系统对话，它们会发展出共同的数学吗？

```
AI-A 的内部几何: 可能是 O(2) 基的
AI-B 的内部几何: 可能是 Weyl 基的

它们对话后：
  - 会收敛到同一个几何？（存在"最优"几何）
  - 会保持不同但能互相翻译？（几何多样性）
  - 会发明新的几何？（涌现）
```

**乐子**：让两个不同架构的 AI 自由对话，看它们是否发展出共同语言。

### 13. AI 发现的新定理

不是 AI 证明人类提出的定理，是 AI **自己发现**新定理。

```
当前: AI 证明 Goldbach 猜想（如果有人提出来）
目标: AI 发现一个没人想到过的数学真理

这需要：
  1. 能提出问题（不只是回答问题）
  2. 能判断"有趣"（不只是判断"正确"）
  3. 能创造新概念（不只是组合旧概念）
```

**乐子**：设计一个"数学创造力"测试，看 AI 能否发明新概念。

```
物理学: 时间是连续的（经典）或离散的（Planck 时间）
计算: 时间是离散的（步骤）或连续的（模拟计算）
AI: 时间是序列位置（离散）还是隐藏状态的流动（连续）

如果时间在不同框架下有不同本质 → 时间的"几何"是什么？
```
